# CentroCultural

¡Bienvenidos al Centro Cultural Ungs!

La aplicación permite a los usuarios de la comunidad explorar, buscar y publicar actividades culturales y educativas en la zona.

Existen dos tipos de interacciones principales:

	* Usuarios no autenticados
	* Usuarios autenticados

Cada uno tiene diferentes capacidades dentro del sistema.

---------- Usuario no logueado (visitante) ----------

Un usuario que no inició sesión puede:

*Explorar centros y talleres
 
    Acceder al listado general de actividades disponibles.
    Visualizar información básica de cada una.

*Buscar centros o talleres mediante:
	Nombre del curso
	Dirección / ubicación

*Interactuar con el mapa
    el usuario puede buscar una zona en el mapa y seleccionar una de las actividades disponibles, la misma se resaltará en el listado de la derecha.



---------- Usuario logueado ----------

Cuando un usuario inicia sesión, se habilitan funcionalidades adicionales.

* Registrar cursos o talleres
El usuario puede:

Crear un nuevo centro/taller, completando datos como:
	Nombre
	Descripción
	Dirección
	Fecha / horario
	Otros campos relevantes
	
* Eliminar un taller registrado por el usuario.
	En caso de querer, el usuario puede eliminar un taller que haya registrado con anterioridad.
		
* Se le habilitará la sección: "Mis talleres"

El usuario tendrá una sección personal donde podrá:
	Ver sus talleres registrados.
	Buscar dentro de sus talleres (Filtrar por nombre o dirección)
	Visualizar un mapa con dichos talleres o centros e interactuar con ellos.


* Se le habilitará la sección:"Inicio"

El usuario puede ver todos los talleres y centros publicados, tanto propios como de otros usuarios.


---------------- Consideraciones generales ---------------- 

* No es necesario estar logueado para explorar o buscar actividades.
* Es obligatorio iniciar sesión para registrar talleres o centros.
* Todos los usuarios pueden acceder a la información del Centro Cultural UNGS:
    Qué es
    Qué actividades ofrece
    Beneficios
    Opción de registrarse en la plataforma
